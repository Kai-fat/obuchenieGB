## Списки

Не нумерованные списки выделяются звездочкой в начале теста (*). Например вот так:
* Элемент 1
* элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
